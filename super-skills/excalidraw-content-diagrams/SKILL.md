# Excalidraw Content Diagrams

Use this for fast, clear diagrams that help explain ideas in videos, posts, lessons, and presentations.

## Workflow

1. Define the concept and audience.
2. Gather grounding context from YouTube, notes, Readwise, script, or user outline if needed.
3. Choose the diagram type:
   - Concept map.
   - Before/after.
   - System loop.
   - Stack/layers.
   - Timeline.
   - Comparison.
   - Funnel.
4. Keep text minimal; the visual should carry the idea.
5. Create shareable/exportable diagram output when tooling is available.
6. Ask for or apply visual refinements: less text, more space, clearer hierarchy.

## Style Rules

- Prefer large shapes and clear labels.
- Keep one idea per panel.
- Use containers/cards for grouped concepts.
- Avoid dense paragraphs.
- Design as if it may appear on screen in a video.
- Include a link or file path to the finished diagram.
