# Paper Canvas Diagrams

Use this for more polished visual canvases than Excalidraw.

## Workflow

1. Identify the canvas goal: explainer, landing page concept, thumbnail board, lead magnet, or brand planning.
2. Use an existing open canvas or create a new one if tooling is available.
3. Ground style from prior visuals, screenshots, brand references, or user notes.
4. Build visual sections with enough spacing for editing.
5. If the canvas updates live, steer early when overlap or cramped layout appears.
6. Export PNG or other usable assets when complete.

## When To Use Paper Instead Of Excalidraw

Use Paper/canvas when the user wants:

- Higher polish.
- Animation.
- Website-like visual systems.
- Editable design boards.
- Thumbnail ideation.
- Lead magnet design.
- Multiple variants.

Use Excalidraw when the user wants speed, simplicity, and educational clarity.

## Review Checklist

- No overlap.
- Text fits.
- Each section has breathing room.
- Visual hierarchy is obvious.
- Animation supports comprehension rather than decoration.
- Export path/link is provided.
